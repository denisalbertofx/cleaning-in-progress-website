# Foto de la CEO

## Instrucciones para agregar la foto de Anisel Perez

1. **Nombre del archivo:** `ceo-anisel-perez.jpg` (o `.png` si es PNG)

2. **Ubicación:** Coloca la imagen en esta carpeta: `/public/images/team/`

3. **Especificaciones recomendadas:**
   - **Tamaño:** 400x400px (cuadrada) o proporción similar
   - **Formato:** JPG o PNG
   - **Peso máximo:** 200KB (optimizada para web)
   - **Calidad:** Como la foto es de baja calidad, mantenerla pequeña (128-160px de ancho) para evitar distorsión

4. **Nota:** La imagen se mostrará en un círculo de 128px (móvil) a 160px (desktop) en la página "Nosotros"

5. **Optimización:** Si la imagen es muy grande, puedes usar herramientas como:
   - TinyPNG (https://tinypng.com/)
   - Squoosh (https://squoosh.app/)
   - ImageOptim (Mac)
