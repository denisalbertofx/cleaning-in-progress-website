# Logo de Cleaning in Progress INC

## Instrucciones para agregar el logo

1. **Nombre del archivo:** `logo-cleaning-in-progress.png` (o `.jpg` si es JPG)

2. **Ubicación:** Coloca el logo en esta carpeta: `/public/images/logo/`

3. **Versiones necesarias:**
   - **Logo principal:** `logo-cleaning-in-progress.png` - Para usar en Header y páginas
   - **Favicon:** `favicon.ico` o `favicon.png` - Para el favicon del navegador (16x16px o 32x32px)
   - **Apple Touch Icon:** `apple-touch-icon.png` - Para dispositivos Apple (180x180px)

4. **Especificaciones recomendadas:**
   - **Logo principal:** 
     - Tamaño: 200x200px o proporción similar
     - Formato: PNG con fondo transparente (preferido) o JPG
     - Peso máximo: 100KB
   - **Favicon:**
     - Tamaño: 32x32px o 16x16px
     - Formato: ICO, PNG o SVG
   - **Apple Touch Icon:**
     - Tamaño: 180x180px
     - Formato: PNG

5. **Optimización:** Usa herramientas como:
   - TinyPNG (https://tinypng.com/)
   - Squoosh (https://squoosh.app/)
   - ImageOptim (Mac)
