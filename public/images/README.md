# Estructura de Imágenes - Cleaning in Progress

Esta carpeta contiene todas las imágenes y assets del sitio web.

## Estructura de Carpetas

```
public/images/
├── hero/              # Imágenes del hero principal (Home)
├── services/          # Imágenes de servicios individuales
├── icons/            # Íconos vectoriales SVG
├── testimonials/     # Fotos de testimonios (opcional)
├── blog/             # Imágenes de portada de artículos
└── before-after/     # Imágenes antes/después (opcional)
```

## Especificaciones por Carpeta

### `/hero`
- **Uso:** Hero principal de la página Home
- **Tamaño recomendado:** 
  - Desktop: 1600x900px
  - Mobile: 1080x1350px
- **Formato:** JPG comprimido
- **Estilo:** Sala/cocina limpia, iluminación natural, moderno minimalista

### `/services`
- **Uso:** Imágenes principales de cada página de servicio
- **Tamaño recomendado:** 1600x900px (16:9)
- **Formato:** JPG comprimido
- **Nomenclatura:** `limpieza-residencial-miami.jpg`, `limpieza-comercial-miami.jpg`, etc.

### `/icons`
- **Uso:** Íconos vectoriales para servicios y elementos UI
- **Formato:** SVG (fondo transparente)
- **Estilo:** 
  - Trazo lineal 2px
  - Esquinas semiredondas
  - Color: #0056A6 (azul primario)
  - Hover: #00A884 (verde acento)
- **Íconos necesarios:**
  - Casa (Limpieza Residencial)
  - Edificio (Limpieza Comercial)
  - Escritorio (Limpieza de Oficinas)
  - Gota/Burbujas (Limpieza Profunda)
  - Alfombra (Limpieza de Alfombras)
  - Casco/Herramientas (Post-Construcción)

### `/testimonials`
- **Uso:** Fotos de clientes (opcional)
- **Formato:** JPG o PNG
- **Estilo:** Rostro neutral, luz suave, fondo blanco
- **Alternativa:** Avatar profesional plano con iniciales

### `/blog`
- **Uso:** Imágenes de portada de artículos
- **Tamaño:** 1200x630px (formato OG social)
- **Formato:** JPG comprimido
- **Nomenclatura:** `post-1.jpg`, `post-2.jpg`, etc.

### `/before-after`
- **Uso:** Comparaciones antes/después (opcional)
- **Formato:** JPG comprimido
- **Estilo:** Realista, no exagerado

## Paleta de Colores

- **Primario:** #0056A6 (Azul Profesional)
- **Acento:** #00A884 (Verde Limpieza)
- **Fondo:** #FFFFFF (Blanco)
- **Texto:** #1A1A1A (Gris Oscuro)

## Notas Importantes

1. Todas las imágenes deben estar optimizadas para web
2. Usar nombres descriptivos y en minúsculas con guiones
3. Mantener consistencia visual en todas las imágenes
4. Las imágenes deben reflejar espacios reales, limpios y modernos
5. Evitar stock photos cliché o con personas posando exageradamente
